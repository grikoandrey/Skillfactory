Итоговое практическое задание в рамках курса Fullstack разработчик на Python.
Игра: "Крестики-нолики" ("Tic-tac-toe")
Описание: логическая игра между двумя игроками на квадратном поле с клетками 3 на 3, один игрок ставит в ячейку "Х" (крестик),
второй ставит в ячейку "О" (нолик). Знаки ставятся в произвольном порядке в пустые ячейки. Первый ход делает всегда игрок, ставящий 
крестик. Игра завершается победой того или другого игрока в случае, когда первый из них заполняет три ячейки одинаковыми символами, 
по вертикали, по горизонтали или по диагонали, то есть одинаковые символы должны быть расположены в ряд. В случае если данное 
событие не наступило, но все ячейки заполнены, то объявляется ничья.
Логика:
игра начинается с выбора, одной из девяти, любой пустой ячейки игроком, который ставит крестик. 
Затем ход переходит к игроку ставящему нолик, в соответствии с аналогичным алгоритмом.
Далее ход снова переходит к игроку "крестик", далее по очереди и так происходит до тех пор:
- три одинаковых символа расположены по горизонтали в одной из трех строк;
- три одинаковых символа расположены по вертикали в одной из трех строк;
- три одинаковых символа расположены по одной из двух диагоналей;
- все ячейки заполнены символами и объявляется "ничья";
Поле представляет собой квадрат с клетками три на три. Следовательно - три строки, три столбца. Внешне выглядит как
простейшая матрица. Следовательно для осуществления хода (постановки символа в пустую ячейку играющему необходимо определить 
номер строки и номер столбца.
Интерфейс: графический интерфейс отсутствует, вся игра происходит в консоли (в данном случае используется интерпретатор PyCharm).
Для наглядности предлагается вывести изображение поля простейшими символами с предварительным подбором расположения.
Upgrade1: внесено изменение, о замене пустых ячеек на цифры от 1 до 9, чтобы игроку было проще и нагляднее заполнять ячейку.
Сооответствено, работа будет происходить с этими цифрами, которые будут заменяться на символы игроков (крестик и нолик).
Upgrade2: дополнительно для визуальной наглядности и логики игры символы игроков окрашиваются в разные цвета, а цифры окрашены в
серый цвет для меньшего восприятия (чтобы не мешали). 
!!! Для чего дополнительно необходимо загрузить библиотеку colorama !!!
На консоль выводится размеченное поле с цифрами, строка - какой игрок производит ход и предложение выбора номера ячейки, в случае если 
вводится иное значение отличное от (1-9) происходит проверка и выводится соответствующее сообщение.
